כדי לשלוח ללקוח קישור לקטלוג, יש להעלות את קובץ ה-HTML ל-GitHub Pages – שירות חינמי שנותן כתובת URL קבועה לקובץ.

📋 הוראות – 4 צעדים פשוטים:
1. כנס ל־github.com וצור חשבון חינם (אם אין לך)
2. צור Repository חדש – שם לדוגמה: marlog-catalog
3. העלה את הקובץ catalog_marlog.html ושנה את שמו ל-index.html
4. בהגדרות Repository ← Pages ← בחר main branch ולחץ Save
✅ הקישור שתקבל יהיה בפורמט:
https://[שם-משתמש].github.io/marlog-catalog/
